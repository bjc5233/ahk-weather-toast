# toast-weather



## 说明
> 获取天气信息，并以系统toast形式展示；建议将脚本配置到系统定时任务中，让其自动执行
---
使用步骤
* 安装autohotkey
* 修改weather.ahk中行localPath变量为自己电脑路径
* 双击nextDay.ahk执行预览
* 在系统计划任务中配置定时执行。
   例如我的配置是：触发器每日9:30，操作"C:\Program Files\AutoHotkey\AutoHotkeyU64.exe" "C:\path\toast\weather\weather.ahk"
